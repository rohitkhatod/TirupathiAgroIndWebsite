# Dependency Categories

## Call Dependency
A source module invokes a target module/subroutine.

## Data Dependency
A source module reads/writes data owned by a target domain.

## Scheduling Dependency
Execution order enforced by batch scheduler or job chain.

## Contract Dependency
Inbound/outbound file/API/message contract coupling.

## Configuration Dependency
Behavior controlled by shared parameters, lookup tables, or control files.

## Operational Dependency
Reliance on restart points, utility jobs, archival, or reconciliation steps.
