# Business Rule Taxonomy

## Validation Rules
Input or state constraints that must hold true.

## Derivation/Calculation Rules
Rules computing target fields from source fields, rates, or lookup factors.

## Eligibility Rules
Rules deciding whether an entity/process can proceed.

## Routing/Workflow Rules
Rules deciding downstream path, queue, or process lane.

## Exception Handling Rules
Rules defining fallback, reject, retry, or escalation behavior.

## Reconciliation/Audit Rules
Rules ensuring balancing totals, audit trail completeness, or consistency checks.
