# Orchestration Playbook

## Stage 0: Intake
- Verify corpus completeness.
- Establish migration objectives and constraints.

## Stage 1: Understanding
- Generate semantic understanding package.
- Approve or flag ambiguities.

## Stage 2: Business Rules
- Produce Word-ready rule documentation.
- Validate traceability and confidence.

## Stage 3: Dependencies
- Build normalized dependency catalog.
- Produce migration sequencing recommendations.

## Stage 4: Visual Architecture
- Generate call graph and flowcharts.
- Validate with SMEs for critical paths.

## Stage 5: Conversion
- Build COBOL conversion specification.
- Generate traceability matrix and quality-gate results.
